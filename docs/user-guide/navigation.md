# Navigation Guide

## Desktop Navigation

On larger screens, the navigation bar displays across the top of the page:

- **Logo**: Click to return to home
- **Section Dropdowns**: Hover or click to see sub-pages
- **Help Icon**: Access documentation
- **Notification Bell**: View unread notifications
- **User Menu**: Sign out and view account info

## Mobile Navigation

On mobile devices:

- **Hamburger Menu** (top-left): Opens the full navigation sidebar
- **Logo** (center): Application name
- **Utilities** (right): Help, notifications, and user menu

### Mobile Sidebar

The sidebar shows all navigation sections as collapsible groups. Tap a section header to expand and see sub-pages.

## Project Context

The project selector in the header filters all views to show only items from the selected project. Select "All Projects" to see everything.

## Active State

The current page is highlighted in the navigation. On desktop, the parent section dropdown also highlights when viewing a child page.

## Keyboard Navigation

- `Tab`: Move between navigation items
- `Enter` or `Space`: Activate buttons and links
- `Escape`: Close dropdowns and sheets
