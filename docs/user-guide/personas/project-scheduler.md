# Project Scheduler Guide

## What You Do

Plan and sequence work packages across projects. Manage PWBS dependencies, create schedules, and coordinate material readiness with logistics.

## Quick Access

- **Primary**: Projects > All Projects - Manage project timelines
- **Secondary**: Projects > Dependencies - View PWBS relationships
- **Also**: Logistics > Arrival Calendar - Material availability

## Key Tasks

- [ ] Define work package sequences and dependencies
- [ ] Adjust schedules based on material availability
- [ ] Coordinate with logistics on critical deliveries
- [ ] Update project timelines when delays occur
- [ ] Communicate schedule changes to stakeholders

## Tips

- Check material readiness before finalizing schedules
- Use the Gantt view for visual timeline management
- Review dependencies before moving work package dates
- Coordinate with multiple projects to optimize resource allocation
