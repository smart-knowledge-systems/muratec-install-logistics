# Site Manager Guide

## What You Do

Oversee field operations at the installation site. Coordinate move-in teams, track inventory status, and ensure parts availability for installers.

## Quick Access

- **Primary**: Supply List > Field Dashboard - Operations overview
- **Secondary**: Supply List > Browse Items - Search for specific parts
- **Also**: Dashboard > EVM Overview - Project health status

## Key Tasks

- [ ] Monitor daily move-in progress on the Field Dashboard
- [ ] Review inventory verification status
- [ ] Coordinate picking priorities with installation leads
- [ ] Escalate material shortages or delays
- [ ] Report daily status to project management

## Tips

- Start each day by reviewing the Field Dashboard
- Use the desktop view for better overview when in the office
- Filter by work package to focus on current installation phase
- Check arrival calendar for upcoming shipments
