# Project Manager Guide

## What You Do

Oversee project health and delivery. Monitor EVM metrics, coordinate across workstreams, and report status to leadership.

## Quick Access

- **Primary**: Dashboard > EVM Overview - Project health metrics
- **Secondary**: Dashboard > Reports - Generate status reports
- **Also**: Projects > All Projects - Multi-project view

## Key Tasks

- [ ] Monitor SPI and CPI daily for early warning signs
- [ ] Review project forecasts (EAC, ETC) weekly
- [ ] Address red-flag items immediately
- [ ] Communicate status to stakeholders
- [ ] Coordinate cross-functional issues

## Tips

- Set up notification preferences for metric thresholds
- Use drill-down to understand specific variances
- Export reports for stakeholder presentations
- Compare multiple projects to identify systemic issues
