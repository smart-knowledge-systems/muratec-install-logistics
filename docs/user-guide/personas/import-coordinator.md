# Import Coordinator Guide

## What You Do

Track shipments from Japan factory through customs to US delivery. Manage milestones, update statuses, and coordinate with freight forwarders.

## Quick Access

- **Primary**: Logistics > Shipment Board - Kanban milestone view
- **Secondary**: Logistics > Arrival Calendar - Delivery schedule
- **Also**: Supply List > Browse Items - Verify shipment contents

## Key Tasks

- [ ] Update shipment milestones as they occur
- [ ] Track customs clearance progress
- [ ] Coordinate with freight forwarders on delays
- [ ] Notify site managers of delivery changes
- [ ] Verify shipment contents against manifests

## Tips

- Check the shipment board daily for status updates
- Use the calendar view for planning site readiness
- Flag potential delays early for contingency planning
- Maintain accurate milestone dates for reporting
