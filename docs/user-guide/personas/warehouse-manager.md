# Warehouse Manager Guide

## What You Do

Manage material storage and staging. Coordinate incoming shipments, organize warehouse space, and ensure parts availability for installation.

## Quick Access

- **Primary**: Logistics > Arrival Calendar - Incoming schedule
- **Secondary**: Supply List > Inventory - Warehouse inventory
- **Also**: Supply List > Browse Items - Search parts

## Key Tasks

- [ ] Prepare staging areas for incoming shipments
- [ ] Coordinate unloading and storage
- [ ] Organize inventory by project and work package
- [ ] Support picking operations for installers
- [ ] Report storage capacity issues

## Tips

- Review the arrival calendar weekly for space planning
- Use case numbers to organize storage locations
- Coordinate with site managers on picking priorities
- Flag damaged or missing items immediately
