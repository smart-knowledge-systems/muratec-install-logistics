# Installer Guide

## What You Do

Install Muratec equipment on site. Receive work packages, collect parts, and track installation progress.

## Quick Access

- **Primary**: Install > Work Package View - Your assigned work
- **Secondary**: Supply List > Picking - Parts for your work package
- **Also**: Supply List > Field Dashboard - Site overview

## Key Tasks

- [ ] Review assigned work packages each morning
- [ ] Collect required parts from staging area
- [ ] Mark items as installed when complete
- [ ] Report issues or missing parts immediately
- [ ] Update progress at end of each shift

## Tips

- Use the mobile interface for hands-free operation
- Scan part numbers to confirm correct items
- Take photos of any damaged or incorrect parts
- Complete installations in work package order when possible
