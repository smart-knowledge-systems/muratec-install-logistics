# Muratec Install Logistics

Welcome to the Muratec Install Logistics system. This application helps manage supply lists, track shipments, schedule projects, and monitor installation progress for Muratec manufacturing equipment.

## Quick Start

1. **Sign in** with your email address
2. **Select a project** from the header dropdown
3. **Navigate** to your primary workspace using the main menu

## Feature Areas

### Supply List

Browse and manage 16K+ supply items. Filter by project, case, pallet, or work package. Mobile-optimized for field operations.

### Logistics

Track shipments from factory to site. Monitor milestones, customs clearance, and delivery schedules using the Kanban board or calendar view.

### Projects

Schedule work packages with PWBS dependencies. View Gantt charts, manage timelines, and track material readiness across multiple projects.

### Dashboard

Monitor project health with Earned Value Management (EVM) metrics. Track SPI, CPI, and forecast completion dates.
